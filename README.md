A simple and lightweight To-Do application built using HTML, CSS, and Vanilla JavaScript.
This project focuses on practicing DOM manipulation, event handling, and local storage without using any frameworks.

---

## 🚀 Live Demo
https://vlasmar.github.io/Vanilla-To-Do-App

---

## 🧱 Features
✅ Add tasks
✔️ Mark tasks as completed
❌ Delete tasks
💾 Save tasks in localStorage
📱 Responsive design

---

## 🛠️ Technologies Used
- HTML – Structure
- CSS – Styling
- JavaScript (ES6) – Logic and interactivity
